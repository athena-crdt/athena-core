## Quickstart
Get started with **athena-core**

```
$ git clone https://github.com/athena-crdt/athena-core
$ cd athena-core
$ go build
$ ./athena-core
```

To run the tests provided

```
$ go test -v -race -count=1 ./...
```

Format using

```
$ gofmt -s -w -l .
```
